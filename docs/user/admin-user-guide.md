# Руководство Пользователя: Admin

## Назначение

Admin-панель предназначена для операционного просмотра данных и удаления чека.

Путь:

```text
/admin
```

## Вход

Для bootstrap-входа должны быть заданы переменные окружения:

- `ADMIN_BOOTSTRAP_LOGIN`
- либо `ADMIN_BOOTSTRAP_PASSWORD`
- либо `ADMIN_BOOTSTRAP_PASSWORD_HASH`

После входа администратор получает web-доступ к:

- поиску чеков
- карточке чека
- операции удаления

## Поиск Чека

На главной странице `/admin` доступен поиск по строке.

В выдаче отображаются:

- название чека
- invite token
- владелец
- chat id
- число участников
- число расходов
- время создания

## Карточка Чека

Внутри карточки чека доступны read-only данные:

- общая информация по чеку
- участники
- история merge
- расходы
- исходные Telegram-тексты расходов
- текущий settlement snapshot

Это позволяет разбирать пользовательские инциденты без доступа к базе вручную.

## Удаление Чека

Удаление является destructive-операцией.

Чтобы удалить чек, администратор должен:

1. открыть карточку чека
2. ввести точное название чека в поле подтверждения
3. отправить форму удаления

Если подтверждение не совпадает, удаление не выполняется.

## Что Важно Помнить

- admin-панель не предназначена для пользовательской работы с расходами
- операции изменения данных через admin ограничены удалением чека
- monitoring доступен как отдельный operational слой через Grafana
